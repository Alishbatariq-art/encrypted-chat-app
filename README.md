# 🔐 End-to-End Encrypted Chat Application

## About
A real-time chat application with end-to-end encryption 
built using Python, Flask, and RSA/AES cryptography.

## Features
- RSA Key Generation (2048-bit)
- AES Message Encryption
- Real-time chat using WebSockets
- Encrypted key exchange between users
- Shows encrypted data on wire

## Technologies Used
- Python 3
- Flask & Flask-SocketIO
- Cryptography Library
- JavaScript WebCrypto API
- HTML/CSS

## How to Run
1. Install requirements:
   pip install flask flask-socketio cryptography

2. Run server:
   python server.py

3. Open browser:
   localhost:5000
